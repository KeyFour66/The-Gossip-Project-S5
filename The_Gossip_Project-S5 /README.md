# The Gossip Project

### THP S5 J1


Construire le front d'une app Rails développée plus tôt
* Liaisons entre les routes, controllers et views

Pour tester l'app :
* Cloner le repo
* `bundle install`
* `rails db:migrate`
* `rails db:seed`
* `rails server`
* Se rendre sur localhost:3000/ dans le navigateur
